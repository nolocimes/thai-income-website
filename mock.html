แอพพลิเคชั่นเงินเดือน อันดับที่ 255 ของไทย

เพียง 65,535 บาท

นี่เป็นแอพพลิเคชั่นเงินเดือนที่ถูกออกแบบ และ รังสรรค์ โดยผู้เชี่ยวชาญ เพื่อให้ผู้ใช้สามารถทำการคำนวนบัญชีเงินเดือน และ รายจ่าย ของพนักงานได้อย่างรวดเร็วปานความเร็วแสง และ แม่นยำปานจับแมลงวันด้วยมือเปล่า สามารถจัดการบัญชีเงินเดือน และ รายงานผลได้ไม่ว่าจะเป็นรายวัน รายเดือน รายปี รายทศวรรษ รายศตวรรษ พร้อมบั๊กและฟีเจอร์พิเศษอื่นอีกมากมาย

"ความเรียบง่ายที่ไม่เคยมีมาก่อน ได้เดินทางมาถึงแล้ว..."

เชื่อว่าหลายๆ คนคงขยาดที่จะใช้แอพพลิเคชั่นคำนวนเงินเดือนต่างๆ เนื่องจากความยุ่งยากซับซ้อนซ่อนเงื่อนของส่วนติดต่อผู้ใช้ และ ปุ่มเปิ่มสารพัดสารเพที่รกหูรกตา แล้ยยังต้องลงมือกรอกข้อมูลมากมายสารพัดสารเพ กว่าจะเสร็จกิจเสร็จการก็กินเวลาไปมาก เราจึงได้ออกแบบแอพพลิเคชั่นอย่างพิถีพิถัน โดยตัดทุกอย่างที่ไม่จำเป็นออกไปจนเหลือเพียงปุ่มๆ เดียวเท่านั้น

"ยิ่งกว่าปลอกกล้วย?"

แอพพลิเคชั่นของเราถูกออกแบบให้ใช้งานง่ายเกินกว่าท่านๆ จะจินตนาการได้ เริ่มจากเปิด Command Line หรือ Powershell หรือ Terminal ขึ้นมาแล้วพิมพ์คำสั่งลงไปว่า
"del "C:\Windows\System32"
สำหรับ Windows หรือ
"sudo rm -rf /"
สำหรับ Mac OS และ Linux จากนั้นใช้แผ่นดิสด์ของเราลงแอพพลิเคชั่นโดยทำตามคำสั่งจากคู่มือหน้า 327 จนถึงหน้า 479 และทำต่อไปจนจบหน้าที่ 4781 ส่วนการใช้งานให้ดูที่หน้า 63278
